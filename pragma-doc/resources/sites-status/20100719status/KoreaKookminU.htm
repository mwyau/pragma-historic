1. Core Services(wsrf based) for MGrid
     - in progress of load & reliability test by KISTI
     - planed to be finished at the end of Aug

2. UI things on the above core services
  a)  Gfarm support - in development(will be release in July)
    - MGrid job can get files from Gfarm and send out results to Gfarm.

  b) sharing VR instances among users - in development (will be released in July)
    - Any user including portal admin can create a VR from physical resources of PRAGMA testbed and let others to use it.

  c) blogging results of MGrid jobs - ready to release
    - You can send the results to any blog site in pre-defined form. ( remote blogging technology)

   d) new UI for login by digital signature - ready to release

   e) new UI for Job building & running - ready to release
    - reduce and minimize steps to two windows

   f) new web application to handle user's preference and SSO - ready to release

   g) new UI for VR - in development (will be released in Aug)

   h) new UI for Workspace - in development (will be released in Sept)
    ** Old stable UIs for VR and Workspace are going to be used with new UI for a while.
    ** All UI are web applications, so you don't have to worry about installing.
