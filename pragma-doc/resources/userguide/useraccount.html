Site Administrators Guide

Site System Interrupt Handling:

  • Scheduling a major downtime with flexible time constraint
    Please contact and discuss with Cindy Zheng, to schedule around any major events

  • When a site system maintenance is scheduled
    the technical contact for the site should email the pragma-grid-support list at least 2 days prior to the scheduled downtime, to inform all supporters
    • the name of the system
    • the start downtime scheduled
    • estimated downtime duration
    • reasons and system changes may impact applications

  • When an unscheduled interrupt (system crash) occurred
    the site technical contact should email the pragma-grid-support list as soon as possible, to inform all supporters
    • the name of the system
    • the time interrupt occurred
    • estimated downtime duration
    • reasons and system changes may impact applications
