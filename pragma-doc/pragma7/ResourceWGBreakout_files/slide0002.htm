Towards a production Grid
Grid technology is still too difficult to use.
In order to solve this problem, ApGrid, PRAGMA, and APAN will collaborate on the following activities:
publishing Grid documents
E.g. Guidelines to set up and operate a grid testbed
running a operational team, namely the Grid Operation Center.
monitoring, providing information about networking/computing resources, trouble shooting, policy management and etc.
supporting application developers
Provides information to implement Grid-enabled applications.
organizing meetings
Use PRAGMA WS and APAN
building knowledge center
documents, training materials, media, news, etc.
running Asia Pacific Grid PMA
define minimum CA requirements
