System Requirement

  • Software requirement
    • Installation of Globus 2.x or 3.x (but not 3.9.x)
      • Build all SDK bundles from the source bundles, with the same flavor
      • Install shared library on both frontend and compute nodes
    • Installation of Ninf-G (Version 2.4.0)
      • Download from here (Installation manual is here.)
      • Use *pthr flavor of the Globus if it is available on your system
    • Installation of any MPI library
    • Installation of any Fortran90 compiler
  • Hardware requirement
    • 2GB Disk storage for data back-up
