TDDFT/Ninf-G


System Requirement

  • Software requirement
    • Installation of Globus 2.x, 3.x or 4.x
      • Build all SDK bundles from the source bundles, with the same flavor
      • Install shared library on the compile node and all compute nodes

    • Installation of Ninf-G (Version 2.4.0, included in NMI 8.0) on the compile node
      • Download Ninf-G and follow the instructions in the installation manual.
      • Use non-thread flavor of the Globus if it is available on your system
      • Don't have to follow these steps in installation
        • Registering the host information
        • Settings for provision of Ninf-G Executable information by MDS

    • Installation of Intel Fortran compiler version 6, 7 or 8
      • Install shared library on the compile node and all compute nodes
      • If version 8 is installed, the build date must be later than May 20, 2004.
      • If version 9 has been already installed on your system, I can test TDDFT with it first.
  • Hardware requirement
    • 300MB disk storage for input data on all nodes (If this is difficult, the data on frontend can be shared by NFS.)
  • Network requirement
    • Compute nodes must connect to outside the internal network via NAT or firewall.
