What is mpiBLAST-g2?

MpiBLAST-g2 is an enhanced parallel program of LANL's mpiBLAST. The enhanced program allows the parallel execution of BLAST on the Grid environment based on Globus Toolkits 2.x and MPICH-g2. The first deployment of mpiBLAST-g2 is on PRAGMA cluster system over the Pacific Rim and was demonstrated on SuperComputing 2004 conference.

[back]

The advantages of mpiBLAST-g2

The mpiBLAST-g2 program has inherent advantages from LANL's mpiBLAST and was improved for sharing resources including computing resources and preformatted BLAST databases in the Virtual Organization on the Grid. The advantages are listed in below:

Inherent advantages from LANL's mpiBLAST:

  • Master-Worker model

  • High throughput

  • Load Balancing

Advantages from the improvements:

  • Sharing computing power in Virtual Organization on Grid, solving problems larger than before.

  • Sharing BLAST databases on remote storage, balancing the loading of local database storage.

  • Security protocol of resource sharing based on Grid Security Infrastructure, protecting the private data.
